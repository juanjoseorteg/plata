Setup Steps (iOS 26)
1
Create a new Automation
Open Shortcuts → Automation tab → tap + → scroll down to find Transaction under Wallet & Apple Pay. Leave all cards and categories checked. Select Run Immediately → tap Next.
2
Search "Get Contents of URL" → fetch accounts & envelopes
URL: https://yzeruqrlgetmihjdsijf.supabase.co/functions/v1/get-envelopes · Expand (∨) → Method: POST → Headers → add key authorization, value = Authorization Header above → Request Body: JSON → add text field token = your token above.
3
Search "Get Dictionary Value" → key: accounts
Key = accounts · "in" = Contents of URL.
4
Search "Choose from List" → account picker
List = Dictionary Value · tap > → prompt: "Which account?"
5
Search "Get Dictionary Value" → key: id (account)
Key = id · "in" = Selected Item (from account list).
6
Search "Get Dictionary Value" → key: envelopes
Key = envelopes · "in" = Contents of URL (the first action).
7
Search "Choose from List" → envelope picker
List = Dictionary Value (envelopes) · tap > → prompt: "Which envelope?"
8
Search "Get Dictionary Value" → key: id (envelope)
Key = id · "in" = Selected Item (from envelope list).
9
Search "Ask for Input" → amount
Type: Number · tap > → prompt: "How much?" · dismiss the variable selector that appears (tap X).
10
Search "Get Contents of URL" → log the transaction
URL: https://yzeruqrlgetmihjdsijf.supabase.co/functions/v1/log-transaction · Method: POST · Headers → add authorization = Authorization Header above · Body: JSON with 4 fields:
token → your token (text)
amount → Ask for Input (variable)
account_id → Dictionary Value id for account
envelope_id → Dictionary Value id for envelope
11
Save ✓
Tap the blue checkmark to save. Done — the automation fires after every Apple Pay payment, asks which account + envelope + how much, then logs it to Plata instantly.
